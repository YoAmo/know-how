# JavaScript: Style

## Usos

 En JavaScript se puede actuar sobre los elementos del DOM y esto incluye la propiedad _Style_

 ### Propieades

 #### backgroundImage

``` js
document.body.style.backgroundImage = "url('img_tree.png')";
```
